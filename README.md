# TJK's Minimap

> "There are many map mods, but this one is yours."

Open-source (MIT) minimap & world map for Minecraft (Fabric, client-only) — built as a **map engine with a public API**. Our own features (radar, cave mode) are addons on the same API you get.

**Status: pre-0.1, in development.** See [VISION.md](VISION.md) · [SPEC.md](SPEC.md) · [PLAN.md](PLAN.md) · [ROADMAP.md](ROADMAP.md)

## For players
One jar: minimap (rotate/north-lock, zoom), fullscreen world map, waypoints + death points, themes, HUD layout — servers can limit radar-style features via a capability handshake.

## For addon devs
Stable-by-1.0 API, single-file API reference (`docs/api.md` / llms.txt), addon template with agent skills, one-paste kickstart prompt. AI-assisted addon dev is a supported workflow, not a dirty secret.
